# 季節の野菜ラインナップ

このドキュメントは、ひだまり農園で販売している季節の野菜、旬の野菜、取り扱い商品について説明しています。

## 現在販売中の野菜（12月〜2月・冬野菜）

### 白菜（はくさい）

冬の定番野菜です。霜に当たると甘みが増し、鍋料理や漬物に最適です。1玉約2〜3kgのずっしりとした白菜をお届けします。

- 価格: 1玉 400円 / 2玉セット 700円
- 保存方法: 新聞紙に包んで冷暗所で保存。カット後は冷蔵庫で保存してください。

### 大根（だいこん）

甘みが強く、みずみずしい冬大根です。煮物、おでん、大根おろし、サラダなど様々な料理に使えます。葉付きでお届けしますので、葉も炒め物などにご活用ください。

- 価格: 1本 250円 / 3本セット 650円
- 保存方法: 葉を切り落とし、新聞紙に包んで冷暗所または冷蔵庫で保存。

### ほうれん草

寒締めほうれん草は、寒さに当てることで糖度が上がり、甘みと旨みが凝縮されています。えぐみが少なく、お子様にも食べやすい味わいです。

- 価格: 1束（約200g） 200円 / 5束セット 900円
- 保存方法: 湿らせたキッチンペーパーで包み、ポリ袋に入れて冷蔵庫の野菜室で保存。

### 長ネギ（ながねぎ）

白い部分が長く、柔らかい下仁田系の長ネギです。焼くと甘みが引き立ち、鍋や焼きネギに最適です。

- 価格: 5本 350円 / 10本セット 600円
- 保存方法: 新聞紙に包んで冷暗所で立てて保存。

### 里芋（さといも）

ねっとりとした食感が特徴の里芋です。煮物、豚汁、芋煮に最適です。土付きのままお届けしますので、日持ちします。

- 価格: 1kg 600円
- 保存方法: 土付きのまま新聞紙に包み、冷暗所で保存。洗うと日持ちしません。
